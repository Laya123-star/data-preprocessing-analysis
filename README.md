## 📘 Project Overview

This project demonstrates how raw data is transformed into a clean and usable format for machine learning.

The workflow includes:

- Data exploration  
- Data cleaning  
- Handling missing values  
- Outlier detection  
- Encoding categorical variables  
- Feature scaling  

---

## 🎯 Objective

The main objectives of this project are:

🔹 Improve data quality  
🔹 Handle missing and inconsistent values  
🔹 Detect and treat outliers  
🔹 Convert categorical data into numerical form  
🔹 Normalize features for ML models

## 🧹 Data Preprocessing Steps

### ✔ Data Exploration
- Checked unique values  
- Performed statistical analysis  
- Renamed columns  

### ✔ Data Cleaning
- Handled missing values  
- Removed duplicates  
- Detected outliers  
- Replaced invalid values (e.g., Age = 0 → NaN)  

### ✔ Data Analysis
- Filtered data (Age > 40 & Salary < 5000)  
- Visualized relationships  
- Counted records by category  

### ✔ Data Encoding
- Applied Label Encoding / One-Hot Encoding  

### ✔ Feature Scaling
- StandardScaler  
- MinMaxScaler
